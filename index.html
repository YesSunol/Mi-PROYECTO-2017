  • conceptos

    Programador Web

    Un desarrollador web debe ser versado en los lenguajes como HTML, CSS y JavaScript, que se consideran básicos para crear un sitio web. De manera usual, los desarrolladores dejan la parte visual a los diseñadores web y se enfocan en crear un sitio con un código limpio. Para ello, toman como referencia el archivo que el diseñador les ha pasado y trasladan ese diseño a código compatible con los navegadores web. Es decir, se aseguran que el sitio se encuentre realmente en la web.

    Diseñador Web

    diseñador web hace uso de programas de diseño empleados por fotógrafos, diseñadores gráficos y especialistas en branding como Adobe Photoshop e Illustrator que utiliza para crear el aspecto visual del sitio web. Es común que el diseño finalizado se guarde en un archivo Photoshop correctamente organizado y nombrado por grupos y capas.

    JavaScript

    Al igual que HTML, Javascript es un lenguaje de programación que se puede utilizar para construir sitios Web y para hacerlos más interactivos. Aunque comparte muchas de las características y de las estructuras del lenguaje Java, fue desarrollado independientemente. El lenguaje Javascript puede interactuar con el código HTML, permitiendo a los programadores web utilizar contenido dinámico. Por ejemplo, hace fácil responder a los acontecimientos iniciados por usuarios (como introducción de datos en formularios) sin tener que utilizar CGI
